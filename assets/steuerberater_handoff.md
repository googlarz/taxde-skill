## Usage Instructions for TaxDE

When generating a handoff package:

1. Fill in `[YEAR]` and `[DATE]` from profile meta
2. Paragraph 1: Draw from `profile.personal`, `profile.employment`, `profile.family`
3. Paragraph 2: List all deductions TaxDE has calculated + confidence level
4. Complexity factors: Be specific — include actual numbers, dates, amounts
5. Questions: Formulate as precise, answerable questions with the user's actual figures
6. Documents needed: Add only what's relevant to the specific complexity factors
7. Already prepared: Pull from the actual calculated breakdown in the session
8. Fee estimate: Use rough StBVV Gegenstandswert table (income + expected refund/payment)

**Fee guidance (rough, 2024)**:
- Simple return (Gegenstandswert <€30,000): €200–€400
- Medium return (€30,000–€80,000): €400–€800
- Complex return (international, equity, business): €800–€2,500+
- US expat specialist: often flat fee €1,500–€4,000 for German + US combined

**Steuerberater vs. Lohnsteuerhilfeverein**:
- Lohnsteuerhilfeverein: cheaper (€100–€200/year membership), but LIMITED SCOPE
  - Only allowed for employees with simple income
  - Cannot handle self-employment, rental >€18,000, foreign income
  - TaxDE handles most of what they handle
- Steuerberater: full scope, higher cost, worth it for complex situations
